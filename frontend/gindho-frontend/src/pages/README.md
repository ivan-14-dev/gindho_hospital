# Pages Layer

Responsabilite:

- Ecrans routables.
- Composition de features.
- Layout local de page.
- Etats de chargement, erreur et vide au niveau ecran.

Regles:

- Les pages restent minces.
- Les pages ne contiennent pas de logique API brute.
- Les pages doivent declarer leurs roles et permissions via le registre de routes.
