# Shared Layer

Responsabilite:

- UI primitives reutilisables.
- Hooks transverses.
- Utilitaires purs.
- Configuration frontend.
- Types techniques non metier.

Regles:

- Ne depend pas de `features` ni de `pages`.
- Ne connait pas les microservices metier.
- Peut exposer des composants accessibles et testables.
